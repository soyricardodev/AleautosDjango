# Proyecto-Ballena

Django 
-Configuraciones de la BD en el archivo BD.py
Comandos desde la carpeta proyectoBallena que contiene el manage.py
pip install -r requirements.txt
py manage.py makemigrations
py manage.py migrate
py manage.py runserver
py manage.py loaddata seed/settings.json

Tailwind
-Desde la carpeta de proyecto proyectoBallena
npm run build
npm run watch
